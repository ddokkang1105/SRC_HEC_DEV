SOAP통신을 위한 IP/Context 정보

  • IdP 가 이중화 되어 SP에서 IdP domain으로 server to server 통신이 어려울 경우 설정
  • domain이 아닌 IdP의 이중화된 ip를 모두 설정한다.(2개 이상 설정되어야 한다)
  • ','를 이용하여 정보를 입력 한다.
  • 예) http://your-idp-domain:port/contextname1,http://your-idp-domain:port/contextname2

세션 유지방식

  • 인증 방식이 세션방식일때 idp의 세션을 유지 시켜 주는 방식을 설정한다.
  • 기본값은 REDIRECT 이다.

세션 유지 호출 URI

  • IDP 세션 유지방식이 REDIRECT 일경우 idp 세션을 유지 시켜주기 위해 호출하는 idp uri

세션 로드밸랜싱 확인시간
초

  • 1초보다 작으면 1초로 설정된다.
  • 단 "SOAP통신을 위한 IP/Context 정보"가 설정 되어있을경우 만 확인을 한다.
  • 기본값은 1초 이다.

세션유지 확인시간
분

  • sp의 session time 보다 15분정도 적게 세팅한다
  • 기본값은 0분 이다.

세션 쿠키명

  • idp session cookie name
  • 기본값은 JSESSIONID 이다.
